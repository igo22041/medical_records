# История изменений

## Версия 2.0 - Расширенный функционал

### Добавлено:
- **Статусы болезней**: Добавлено поле статуса для медицинских записей
  - Активная
  - На лечении
  - Выздоровел
  - Хроническая
  - Отменена

- **Фильтрация по статусу**: Возможность фильтровать записи по статусу для пользователей и администраторов

- **Расширенная статистика для администратора**:
  - Распределение записей по статусам с процентным соотношением и визуализацией
  - Статистика активности пользователей (количество записей каждого пользователя)
  - Распределение пациентов по возрастным группам
  - Список последних добавленных записей

- **Улучшенный интерфейс**:
  - Цветовые индикаторы статусов
  - Визуальные графики статистики
  - Улучшенная панель администратора

### Изменено:
- Обновлена модель `MedicalRecord` для поддержки статусов
- Обновлены формы добавления и редактирования записей
- Улучшена панель администратора с детальной статистикой

### Технические детали:
- Добавлено поле `status` в таблицу `medical_records`
- Создана миграция `migration_add_status.sql` для обновления существующих БД
- Добавлен файл `config/statuses.php` для управления статусами

## Версия 1.0 - Базовая версия

### Функциональность:
- Регистрация и авторизация пользователей
- CRUD операции для медицинских записей
- Поиск записей
- Панель администратора
- Базовое управление записями
